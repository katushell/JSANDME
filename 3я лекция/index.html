DOM

узел - любой элемент дерева

родитель - элемент, из которого растёт узел

дети - элементы, которые растут из узла

корень - элемент из которого растёт дерево, элемент без родителей

лист - элемент дерева, который не имеет детей

Документ - объект, который описывает открытую в браузере страницу и он явл корнем того дерева, которое описывает всю разментку, которая есть на странице

доступ идёт через body, потому что это самый быстрый способ

первым потомком body явл НЕ HTML, первым эл DOM дерева будет объект, который описывает текст разметки, который содержит перенос и табуляцию до этого заголовка

body -tab\t -h1 -tab\t -a -Link text - текст ссылки описывается, как отдельная нода (node-узел)

Что можно делать с деревом?

(Depth-First Search) поиск в глубину

(Breadth-First Search) поиск в ширину

поиск в ширину исп для отрисовки дереа поиск в глубину для поиска конкр элемента

коллекция - это псевдомассив

document.getElementById('idName') в document есть коллекция, которая называется all в ней лежат все элементы со страницы в линейном виде элементы записаны по id и поиск через id залезает туда

есть метод querySelector('CssSelector'); использует поиск в глубину можно искать по классам, айди и даже просто тегам так же можно исп вложенности и ключи css (active, focus etc)

Дима самый лучший

псевдоэлементы не явл частью DOM

querySelectorAll('CssSelector'); покажет ВСЕ элементы которые соответст селектору

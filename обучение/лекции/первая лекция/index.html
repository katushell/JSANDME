Типы данных

Примитивные

Те данные, про которые можно сказать чему они равны

Числа Строки (текст) Логические (булевы значения) служебные (null, undefined)

Числа

Записываются без дополнительных символов

дробная часть отделяется точкой (!только точкой!)

доп значения : Infinity (бесконечность); NaN (не мат операция)

можно записывать в 10тиричном и 16тиричном формате (перед тем как писать в 16тиричном нужно написать 0x например: 0xFF)

Строки

'Это строка' ; "Это тоже";

пробел и перенос строки это символы т е называются строками, потому что переносы, пробелы и отступы явл обычными символами, поэтому весть текст хранится в виде одной длинной последователньости символов

в случае, если в стрке нужно использовать спец символ, используется экранирование "\tHello, I\'m a string\n\n"

Оператор

Оператор - это минимальная единица языка, команда. Оператор принимает несколько значений, преобразует их и возвращает новое значение"

унарные (операторы, которые работают с одним значение) (+ , - )
бинарные (операторы, кот работают с двумя значениями)
тернарные (единственный в языке оператор, который принимает на вход три значения)

математические строковые оператор логические операторы (операторы сравнения, операторы булевой логики) служебные операторы

конкатинация - сложение текстов ('Дима' + ' Самый' + 'Любимый') получится (правда)

Приведение типов

если операнды, переданные операторы разного типа, они автоматически приводятся к типу, в котором операцию можно выполнить. Тип, в который будут приведены операнды зависит от оператора

console.log ( '10' / 2 ); //5

console.log ( '10' + 2 ); //102 у строкового сложения приоритет над математическим

console.log ( 'true' + eee ); //trueeee строковый оператор - приведение производится к строке

Явное приведение

приведение к числу parseInt('10 , 10') === 10; parseFloat ('10.2') === 10.2;

приведение к строке 10.toString(); // ошибка, потому что js умеет работать с дробными значениями и ждёт после точки дробную часть, а там слова тустринг 10..toString() === '10'; 10.0.toString() === '10';

приведение операторами

приведеение к числу с помощью оператора знака ( унарный + или унарный - )

+'10' === 10; -'3' === -3;

Приведение к строке сложением с пустой строкой Т к приоритет у строкового значения 3 + "" == "3";
